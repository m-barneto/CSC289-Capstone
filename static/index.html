Instructions

Welcome to the Assignment Tracker! This is the main page, where you can find any and all functions of this program. The options at the top allow you to view your calendar, as well as save any changes you have made. You can even export your calendar and use the file on a different device by importing it. The Main Menu button will bring you back to this page. These options will be available on every page in the same location. To the left you will see your options for adjusting your calendar. The options are pretty self-explanitory. Add Assignment allows you to add a new item to your calendar, Edit Assignment is used for editing items already on your calendar, and Remove Assignment is for deleting items from the calendar. If you want to keep track of finished assignments, make sure to mark them as completed, instead of deleting them.
